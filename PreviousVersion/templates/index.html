Welcome to the QR Code Generator Backend

This application serves APIs for generating customized QR codes.

API Endpoints

  • POST /api/advanced/qr: Generate a customized QR code

API Usage:

Send a POST request with the following JSON payload:

{
  "text": "Your text here",
  "fill_color": "blue",
  "back_color": "white",
  "module_shape": "default",
  "eye_shape": "default",
  "gradient_type": "solid",
  "logo_url": "https://example.com/logo.png"
}
